# Style Guide – Palette and Typography

This style guide captures the shared visual language for MindLore. Use it as the single reference for colours, typography, and basic spacing.

## Colour Palette

- Copper (accent): `#B87333`
- Turquoise (primary action): `#30D5C8`
- Verdigris (primary alt): `#43B5A0` or softer `#679267`
- Charcoal / Gunmetal (text/anchor): `#4A4A48` or `#2C2C2C`
- Ivory / Champagne (background): `#F9F6EF` or `#E8D8C3`
- Amber / Brass (highlight): `#FFBF00` or `#D4A017`
- Midnight / Prussian (contrast): `#003366` or `#1F4E5F`

Usage:
- Use verdigris/turquoise for primary surfaces and calls to action.
- Use copper/amber sparingly for highlights and icons.
- Pair charcoal or midnight text over ivory/champagne backgrounds for legibility.

## Typography

- Headings: Lora, serif; weights 500–700.
- Body/UI: Inter, sans-serif; weights 400–600.
- Base size: 16px; scale with rems (e.g., 0.75rem, 0.875rem, 1rem, 1.25rem, 1.5rem).

## Spacing and Radius

- Spacing scale (rem): 0.25, 0.5, 0.75, 1, 1.5, 2.
- Corners: small radius 4px for chips/buttons; medium 8px for cards/panels.

## Notes

- Prefer CSS variables or token imports to ensure consistency.
- Keep accents minimal to avoid overwhelming the base palette.
